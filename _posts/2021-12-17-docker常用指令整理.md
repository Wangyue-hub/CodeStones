整理docker的常用指令,用于快速查阅使用：

# 镜像

获取本地镜像集合
>docker image ls

拉取镜像
>docker pull repository/image:ver

移除镜像
>docker image rm -f imageId

运行镜像
>docker run repository/image:ver

打包镜像
>docker save -o imagename.tar imagename:ver

载入镜像
>docker load -i image.tar
